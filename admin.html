▶ GitHub Repository Config ⚪
Вкажіть ваш репозиторій, щоб зміни зберігалися напряму в data.json через GitHub API.

Owner — ваш GitHub username або організація.
Repo — назва репозиторію (наприклад CoctailHelper).
Branch — зазвичай main або gh-pages.
Path — шлях до файлу в репо, зазвичай data.json.

Ці налаштування зберігаються лише у вашому браузері (localStorage).
